# GrowthDesk AI Beta Launch Guide

## Goal
Prepare the SaaS for a controlled beta launch with real agency users.

## Launch Checklist
- Configure production environment variables.
- Run database migrations.
- Create first super admin account.
- Create demo tenant and sample data.
- Verify tenant isolation manually.
- Test registration, login, clients, leads, projects, invoices, AI tools, tickets, and client portal.
- Invite 3-5 beta agencies first.

## Recommended Beta Offer
- 14-day free trial.
- Founder pricing for first 20 customers.
- Collect feedback weekly.

## Test Accounts
Create separate accounts for:
- Super Admin
- Tenant Owner
- Manager
- Staff
- Client Portal User
