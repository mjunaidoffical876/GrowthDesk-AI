# GrowthDesk AI Launch Checklist

## Before beta launch
- [ ] Production database configured
- [ ] JWT secret changed
- [ ] OpenAI key configured
- [ ] Email provider configured
- [ ] Stripe/payment provider configured
- [ ] Domain connected
- [ ] SSL enabled
- [ ] Error monitoring enabled
- [ ] Backups configured
- [ ] Demo tenant tested
